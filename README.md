 
# Travel Website
A simple, responsive travel-themed website built using HTML, CSS, and JavaScript.  
This project showcases different travel locations, UI layouts, cards, navigation bars, images, and animations commonly used in modern travel websites.

## Features
- Clean and responsive UI
- Hero section with background image
- Destination cards and details
- Navigation bar with links
- Sections for tours, packages, and offers
- Smooth scrolling and basic animations
- Mobile-friendly layout

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (basic interactivity)

## Folder Structure
/travel-website
│── index.html
│── style.css
│── script.js
│── /images
│── /assets

## How to Run
1. Download or clone the project
2. Open index.html in your browser
3. That’s it. No installation required.

## GitHub Pages (optional)
To deploy:
- Go to Settings
- Scroll to Pages
- Choose branch: main
- Folder: /root
